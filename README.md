# Bloomy Orchard

Sweet pickings and serve delights!

## Installation

Clone the project

```bash
  git clone https://github.com/hailinzparra/bloomyorchard.git
```

Go to the project directory

```bash
  cd bloomyorchard
```

Install the project

```bash
  npm run init
```

## Run Locally

Start development mode

```bash
  npm run dev
```
